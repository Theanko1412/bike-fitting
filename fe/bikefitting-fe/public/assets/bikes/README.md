# Bike diagram assets

Each **bike type** has its own **model** folder under `public/assets/bikes/`:

| Type     | Folder (see `BIKE_DIAGRAM_ASSET_SUBFOLDER` in code) |
|----------|------------------------------------------------------|
| Road     | `road/tarmacSL8pro/masked/`                          |
| Gravel   | `gravel/s-worksTurboCreo/masked/`                  |
| Mountain | `mountain/s-worksEpic8/masked/`                    |

Files: `side.png`, `front_right.png`, `back_right.png`

Rename a model folder or edit `src/config/bike-diagram-assets.ts` if you add another bike.
